# CRUD-APPLICATION
A RESTful CRUD API built with Node.js, Express, MongoDB. Supports, create, read,update and delete operations.Tested with POSTMAN.
The API allows users to perform full database operations through RESTful endpoints.
TECH STACK: Node.js Express.js MongoDB mongoose and POSTMAN for API testing.
FEATURES: create new data, read or GET data, Update existing data, delete data, RESTful API structure, Connected to MongoDB database.
API Endpoints example: POST/GET/PUT/DELETE.
